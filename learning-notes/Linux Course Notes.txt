|	/dev/ >- All the Devices Folder/ All references to CPU.

|	/etc/ >-System Config. files ETI Password Files.

|	/boot/ >- Startup Files and Karnel. Where Current Karnel lives.
+----------------------------------------------------------------------------------------------------------------------------------------+
												!! IMPORTANT !
|	!! /bin/ !!  >- BinaryFiles, Common Files shared by system by it self.
+----------------------------------------------------------------------------------------------------------------------------------------+

|	/misc/ >- Miscellaneous files. Files that your system might saved.

+----------------------------------------------------------------------------------------------------------------------------------------+
												!! IMPORTANT !
|	!! /mnet/ !!  >- Mount Devices/ Works like a CD/DVD that you put in Windows. But Linux don't do that automatically. U have to do it manually. Or a USB or something. Default Folder.
+----------------------------------------------------------------------------------------------------------------------------------------+


|	/lib/ >- Library files, includes files for all kinds of programs need by the system or the user.
+----------------------------------------------------------------------------------------------------------------------------------------+
												!! IMPORTANT !
|	!! /home >- Home Directory. User JohnDoe /home/JohnDoe !!

|	!! /lost+found >- Files that are saved during failure are stored here. Every partition has a lost+found upper dir. !!
+----------------------------------------------------------------------------------------------------------------------------------------+


|	/net/ >- Standard mouth point for entire remote file system.

|	/opt/ >- Containt Extra and Third party software/ Something like the programs that you install from Software Programs, VSCode, SublimeText etc.

|	/proc/ >- A virtual file contain info about system resources.

|	/root/ >- The admin user s home directory. Mind the difference between  / and  /root. The home directory of the root user.

|	/sbin/ >- Programs use by system and sysadmin.

+----------------------------------------------------------------------------------------------------------------------------------------+
												!! IMPORTANT !!
|	!! /tmp/ >- Temp space for the user of system. All the files are cleaned after reboot. Don't use it to store files. !!

|	!! /user/ >-  Program/ Libraries, Documents for all users. !!

|	!! /var/ >-  Storage all var and temp files created by user. !!

|

|	tracert/traceroute Command >- Used to determine more infomation about the path o the destionation host.

|		** Tells us about how many hops are done there, that are by-passed between routers to get that ip address.
